MarketSync Privacy Policy

Effective Date: May 27, 2026

This Privacy Policy describes how MarketSync ("we", "our", or "us") collects, uses, and discloses information when you use our Chrome Browser Extension (the "Extension"). We are committed to protecting user privacy and ensuring complete compliance with the Google Chrome Web Store Developer Program Policies, including the Single Purpose and Limited Use rules.

1. Single Purpose & Scope

MarketSync is built with a single, narrow purpose: to automate inventory cross-listing by parsing active product details from user-selected source inventory tabs and localizing/formatting that data to sync directly into destination marketplace forms. This policy applies strictly to data processed via the Extension.

2. Information We Collect and Process

To fulfill its single purpose, the Extension accesses and processes the following data categories under strict authorization limits:

  • Website Content: When explicitly triggered by clicking the extension icon, MarketSync parses text, images, and structure metrics (DOM data) from your active tab (e.g., source inventory systems or e-commerce pages) to generate matching inventory profiles.
  • Authentication Information: If you connect the Extension to a central MarketSync account dashboard, encrypted session credentials or API tokens are securely managed to maintain your authenticated state.
  • Local Storage Data: The extension utilizes local browser storage (`chrome.storage`) to save user configuration preferences, custom field mapping keys, and local operation logs.

3. How We Use Information

We process your data strictly to execute requested features. Specifically, data is utilized to:

  • Map relevant inventory text and images into your targeted e-commerce or listing marketplace forms.
  • Maintain your localized custom settings and user authentication credentials.
  • Generate exportable sync logs and local CSV/JSON inventory backups when explicitly requested via the user interface.

4. Google Chrome Web Store Compliance Certifications

In accordance with the Chrome Web Store Developer Program Policies, MarketSync certifies the following handling mandates:

  • No Sale or Transfer: We do not sell or transfer user data to third parties, outside of the approved operational use cases explicitly required to fulfill the syncing action.
  • Purpose Limitation: We do not use or transfer user data for any purposes completely unrelated to the core single purpose of cross-listing automation.
  • No Credit Scoring: We absolutely do not use or transfer user data to determine consumer creditworthiness or for lending evaluation purposes.

5. Data Retention and Security

Data parsed dynamically from web pages via the `activeTab` and `scripting` APIs is ephemeral and handles operations in short-lived runtime memory. It is not permanently cataloged or cached on external remote infrastructure unless you explicitly initiate an external cloud sync profile. Local adjustments stay stored securely within your local sandboxed Chrome profile via the browser’s native security frameworks.

6. Third-Party Websites and API Targets

The Extension interacts directly with third-party marketplaces (such as Facebook Marketplace, eBay, or other e-commerce targets) via automation scripts or user-configured APIs. This Privacy Policy governs only MarketSync’s data operations. We encourage you to review the privacy protocols and developer agreements of any recipient platform where you execute listings.

7. Changes to This Policy

We may modify this privacy policy periodically to reflect operational, operational-security, or regulatory adjustments. Any updates will be pushed directly within new extension builds and posted transparently to our Chrome Web Store listing page.
